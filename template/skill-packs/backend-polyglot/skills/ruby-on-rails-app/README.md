# Ruby on Rails App Skill

Optional Garda skill for Ruby on Rails backend applications.

Use it when the repository contains:
- Rails controllers, models, and Active Record migrations
- API endpoints with serializers (ActiveModelSerializers, Jbuilder, Alba)
- Background job processors (Sidekiq, GoodJob, Solid Queue)
- Policy-based authorization (Pundit, Action Policy)
